# PSUSphere

A web application for managing student organizations at PSU.

## Features
- Manage colleges and academic programs
- Register student organizations
- Track student membership in organizations
- Admin interface for data management

## How to Run
1. Clone the repository
2. Create a virtual environment: `python -m venv psusenv`
3. Activate it: `.\psusenv\Scripts\Activate.ps1`
4. Install dependencies: `pip install -r requirements.txt`
5. Run the server: `python manage.py runserver`
